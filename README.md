## Installation

```bash
git clone https://github.com/RhysLees/AceOfSpades.git
```

```bash
cd AceOfSpades
```

```bash
composer install
```

```bash
cp .env.example .env && php artisan key:generate 
```

```bash
npm install && npm run build
```

```bash
php artisan native:install
```

```bash
php artisan native:serve
```
