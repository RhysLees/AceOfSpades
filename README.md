# AceOfSpades

## Installation

```bash
git clone https://github.com/RhysLees/AceOfSpaces.git
```

```bash
cd AceOfSpaces
```

```bash
cp .env.example .env && php artisan key:generate 
```

```bash
composer install
```

```bash
npm install && npm run build
```

```bash
php artisan native:install
```

```bash
php artisan native:serve
```
